# My dotfile
I manage it using [stow](https://www.gnu.org/software/stow/), it is batch symlink creating tool.
It is a interesting tool, give it a try if you are looking to manage your dotfile.
Previously i used a script to copy files from everywhere into a git folder, after
that i got to know about git bare repository. And now this.

## My setup
- Arch linux   OS
- i3           WM
- ewwbar       bar
- picom        compositor
- kitty        terminal emulator
- neovim       editor
- tmux         terminal multiplexer
- cmus         music player
- zen          browser
- ranger       file manager
